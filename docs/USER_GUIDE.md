# Οδηγός Χρήσης – Katoptron Tennis Forehand Analyzer

## Βήμα-βήμα χρήση

1. **Άνοιξε την εφαρμογή**  
   → http://127.0.0.1:8000/ (τοπικά)

2. **Πάτα «Upload»** (ή πήγαινε κατευθείαν στην αρχική σελίδα)

3. **Ανέβασε δύο βίντεο**

   | Θέση          | Περιεχόμενο                               | Προτάσεις ποιότητας                     |
   |---------------|--------------------------------------------|------------------------------------------|
   | Reference     | Professional / ιδανικό forehand (π.χ. Federer, Alcaraz) | πλάγια κάμερα, καθαρό background, 30+ fps |
   | Comparison    | Το δικό σου forehand βίντεο               | ίδια γωνία κάμερας με το reference       |

   → Μέγιστο μέγεθος αρχείου: 100 MB ανά βίντεο

4. **Πάτα «Start Comparison»**

5. Περίμενε 1–5 λεπτά (ανάλογα με μήκος βίντεο & υπολογιστή)

6. **Δες τα αποτελέσματα**

   Θα δεις:

   - **AI Coaching Report** (από GPT-4o)
     - Συνολική εκτίμηση επιπέδου
     - Κρίσιμα λάθη & κίνδυνος τραυματισμού
     - Συγκεκριμένες διορθώσεις ανά φάση
     - Προτεινόμενα drills
     - Πλάνο προπόνησης (άμεσο + μεσοπρόθεσμο)

   - **Συγκριτικό βίντεο**  
     Κόκκινος σκελετός → reference  
     Πράσινος σκελετός → εσύ  
     Κίτρινα σημεία → μεγάλες αποκλίσεις

   - **Μετρικές**
     - Average position difference
     - Average angle difference
     - DTW distances
     - Πλήθος ευθυγραμμισμένων frames

7. **Κατέβασε ό,τι χρειάζεσαι**

   - comparison.mp4 (το overlay βίντεο)
   - landmark_statistics.csv
   - joint_statistics.csv
   - summary.txt
   - pose_differences_complete.json

## Συμβουλές για καλύτερα αποτελέσματα

- Και τα δύο βίντεο να έχουν **παρόμοια γωνία λήψης** (ιδανικά side-view)
- Προτίμησε **slow-motion** ή κανονική ταχύτητα ≥ 60 fps
- Φρόντισε να φαίνεται **όλο το σώμα** (από κεφάλι μέχρι πόδια)
- Απόφυγε πολύ φαρδιά ρούχα – εμποδίζουν την ανίχνευση
- Κράτα **σταθερή κάμερα** (τρίποδο)

## Συχνές Ερωτήσεις

**Q:** Γιατί βγάζει error "No valid pose data";  
**A:** Κάποιο από τα βίντεο δεν έχει καθαρή ανίχνευση σώματος → δοκίμασε καλύτερο φωτισμό ή πιο καθαρό background.

**Q:** Πόσο κοστίζει η AI ανάλυση;  
**A:** ~0.02–0.05€ ανά ανάλυση με GPT-4o (εξαρτάται από μήκος prompt & απάντηση)

**Q:** Μπορώ να το τρέξω χωρίς OpenAI κλειδί;  
**A:** Ναι – η βασική σύγκριση (video + μετρικές) δουλεύει. Η προπονητική ανάλυση θα λείπει.
